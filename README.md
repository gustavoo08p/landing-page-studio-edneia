# Studio Edneia Braz | Landing Page

Landing page responsiva desenvolvida para um salão de beleza feminino, com foco em conversão, identidade visual elegante e contato rápido via WhatsApp.

## Sobre o Projeto

Este projeto foi criado para funcionar como a presença digital inicial do Studio Edneia Braz, apresentando os serviços, reforçando a confiança da marca e conduzindo a cliente até o agendamento com o menor atrito possível.

A proposta combina:

- visual moderno e acolhedor
- navegação simples e objetiva
- prova visual com resultados reais
- estrutura leve e pronta para evoluir

## Funcionalidades

- seção hero com chamada principal e CTA de agendamento
- área de serviços com apresentação clara dos atendimentos

- seção de contato com WhatsApp, Instagram e endereço
- CTA final para reforçar conversão

- animações leves com JavaScript puro

## Decisões de Implementação

- HTML5 semântico para melhor organização e manutenção
- CSS com Grid e Flexbox para responsividade
- imagens locais para maior controle do layout
- CTA distribuído em pontos estratégicos da página
- JavaScript leve, usado apenas para aprimoramento visual

## Objetivo de Negócio

O principal objetivo da landing page é gerar agendamentos de forma rápida e prática, especialmente para clientes que chegam pelo Instagram, indicação ou campanhas locais.

Em vez de depender de um sistema complexo logo no início, o fluxo foi simplificado para:

1. apresentar o valor do serviço
2. mostrar resultados visuais
3. gerar confiança
4. levar a cliente ao WhatsApp

## Como Executar Localmente
